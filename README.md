# 🛒 Shri Jain Stationery Mart – E-Commerce Web Application

A modern, responsive e-commerce platform built to digitize the operations of **Shri Jain Stationery Mart**, a traditional stationery store located at Arya Samaj Road, Kota, Rajasthan. The application provides a seamless online shopping experience with secure payments, automated order processing, inventory management, and a powerful admin dashboard.

🌐 **Live Demo:** [Shri Jain Stationery Mart](https://shri-jain-stationery-mart.onrender.com)

---

## 📌 Features

### 👤 Customer Features

* Browse stationery products through a categorized catalog
* Product search and filtering
* Add-to-Cart and Buy-Now functionality
* Secure checkout and address management
* User registration and authentication
* Order tracking and order history
* Responsive design for desktop, tablet, and mobile devices
* Secure payment integration (UPI, Credit/Debit Cards, Net Banking – Sandbox)

### 🛠️ Admin Features

* Centralized Admin Dashboard
* Product, Category, and Brand Management
* Inventory Management with real-time stock updates
* Low Stock Alert System
* Coupon and Discount Management
* Order Processing and Tracking
* Revenue Dashboard and Sales Analytics
* Product Performance Monitoring
* Customer Management
* Audit Logs for administrative activities
* Easy Product Onboarding and Catalog Updates

### 🤖 Automation

* Automatic order storage in Google Sheets
* Email notifications for customers and administrators
* Automated order confirmation workflow
* Delivery status updates via email
* Automatic inventory updates after successful orders
